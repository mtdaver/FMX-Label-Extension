This Google Extension allows you to print directly from FMX instead of downloading a document from FMX and then printing 
your label(s). 

There are currently Six options:

1. Student - This will print an individual student label.
   a. If you 1-to-1 to look up your device click on the device tag number and you can print directly from here.
   b. If you scan the QR code then you will need to scroll to the bottom of the page and press the "Back" button
      you can then print the single student label.

2. Staff - This will print an individual Staff Label
   a. The method is the same as above.
   b. **WARNING** Not all Staff have the Model Year, and Full Label Name filled in.
      If this is the case then click the edit button and fill in the missing fields.

3. iPad Student
   a. This option work as above, but currently not all iPads have been updated in FMX.

4. iPad Staff
   a.his option work as above, but currently not all iPads have been updated in FMX.

5. Bulk (By Class/Grad Year) - This option will allow you to bulk print from the Equipment page in FMX.
   a. Navigate to the Equipment page in FMX
   b. Click the Filter icon and select: Equipment Type: Technology> Chromebook
   c. In the upper right corner of your screen click on the Settings Icon and select the following:
      i. QR Code
     ii. Full Label Name (Drag this item to the top of your settings it makes it easier to view, and sort)
    iii. Model Year
     iv. Graduation Year
      v. Serial Number - Technology
   d. In the search box: Type in the desired class Graduation year
   e. Sort by the Assigned Users column to everything in alphabetical order
   f. Print your labels. If you have undesirable, or un-needed labels Edit the devices in question
      and remove the Graduation year.
   g. **WARNING** This extension can only read one page at a time. You will navigate to the subsequent pages to print
      those labels.
   h. Need to reprint a label? Put a check mark in only that device, and it will only print that label, or lablels

	 6. Bulk (Custom) - This option will allow you to bulk print from the Equipment page in FMX.
      a. You will need to filter, and choose the same settings as above.
      b. Follow the prompts
      d. Select 1 to use the Full Label Name
      e. Select 2 to create your own Full Label with an index number
         i. Create your Full Label Name: MUL-Summer CB  (there is no reason to put a '-' here as the index option does this.
        ii. Select a starting index ppoint 1 is default. This will append to your Full Label Name -01
      f. **WARNING** This extension can only read one page at a time. You will navigate to the subsequent pages to print
      those labels.
      g. Need to reprint a label? Put a check mark in only that device, and it will only print that label, or lablels
